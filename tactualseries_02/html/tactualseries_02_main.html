dot . a scene = sinθ at the sea _ tactuaL [si:gak] series is a project presenting a communicational method for the visually handicapped to the non-handicapped using physical computing. We can read a sentence by Hands.
Braille Type >>> Visual Type

Tactual Sense + Light = Text 		We can see a picture by Hands.
Braille Drawing >>> Visual Drawing

Tactual Sense + Light = Image
 We can listen a song by Hands.
Braille Listening >>> Visual Listening

Tactual Sense + Listening = Sound
This study focuses on how to read textual expressions by using tactual & visual senses. Each installation works by an input signal from a touch sensor. Spectators can see texts and pictures, and hear the sounds by tactual input behavior.
Through this interactive process,
we can understand how the blind may feel.
